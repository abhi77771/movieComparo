This Project helps us to Compare the Release Date(India Location) and the Extract Country of the Movie Pushpa:The Rise Part 1 
from two different Websites i.e. Imdb.com and Wikipwdia.com

To run the Script, follow the following steps:
1)Open the Project in Eclipse either from Git pull or by downloading a zip file and then extracting the file and opening the project in eclipse 
2)Right click on the peoject and clicks on Maven -> Update Project
3) Click on src/test/resources --> Features --> Movie_comparo.feature
4) Right click on the feature file and click Run As --> 1 Cucumber feature
5) the Script starts getting executed.
6) you can find the logs getting registered in the console.
